HOW TO PUBLISH (NETLIFY DROP)
1) Go to Netlify -> Sites -> Add new site -> Deploy manually (Netlify Drop)
2) Drag & drop this ZIP file directly, or unzip and drop the folder.
3) After deploy: Site settings -> Domain management -> Add custom domain: hilalaydogdu.com
4) In Namecheap, set DNS to Netlify as instructed (Netlify will show exact records).
